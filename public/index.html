REFLECTION

I had fun working my way through this course. It was more challenging than I thought it would be. Most of my prior experience with web dev was building React / Node apps, so getting back to the basics and writing vanilla JS, HTML, and CSS without libraries / frameworks was good for me.

I definitely made things harder on myself by electing early in the semester to deviate somewhat from the course recommendations and use some of my own tooling and infrastructure to support the site. I learned a lot doing things this way, but it did distract me a bit from the core course content.

Overall, I had a good time and am grateful for the experience!
